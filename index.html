Veja porque fazer seu site comigo

1. Negociação

Nessa fase irei entender qual objetivo do site, tirar duvidas, verificar quais os recursos terão disponíveis, prazos de entrega e formas de pagamento.

2. Texto / copy

Para continuar prosseguindo na criação do site é necessário o fornecimento da copy. Caso não tenha, você poderá contratar a parte.

3. Criação / Design

Após confirmação da primeira parcela, textos, imagens em mãos darei inicio a criação do layout do site conforme combinado.

4. Apresentação do site

Nessa fase irei apresentar seu novo site, onde você vai verificar se estar de acordo e caso contrario pedir as alterações necessárias.

5. Site aprovado

Com site aprovado é hora de transferir para hospedagem que você contratou, a partir desse momento seu site estará na internet.

6. Com site publicado

Com o site publicado irei realizar alguns testes com a utilização de plugin para otimizar e deixá-lo mais rápido possível.
